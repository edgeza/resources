Installation:
1) Insert folder "bzzz_pot" to resources folder
2) Add to server.cfg
3) Restart server


--------------------------------------------------
If you STREAM props in ANOTHER resource, you must edit the fxmanifest.
Add this line:
data_file 'DLC_ITYP_REQUEST' 'stream/bzzz_bzoombox.ytyp'

Then the server must be restarted. 
YTYP loads properties of props.
