# Installation Guide

## To Enable Interaction Instead of Target

### [GitHub Repository](https://github.com/darktrovx/interact)

1. Click on the **Code** button.
2. Choose **Download ZIP**.
3. After the download finishes, extract the folder to your server files.
4. Remove the "-main" suffix from the folder name.

That's it! You're all set!
