INSTALLATION:

1) Drag and Drop the resource "cfx-bigoz-mapdata" to your resources folder.

2) Start the resource inside your config.cfg, and make sure to put that above other BigoZ's maps
    `ensure cfx-bigoz-mapdata`

3) Clear your Server Cache

4) Restart your Server

5) Done!
