or 
- qb-target https://github.com/qbcore-framework/qb-target

Installation Instructions:

1. Resource:

Place the 'fh_towing' folder into your resources directory.
Start the script either by starting the whole director like e.g. 'ensure [standalone]' (if the script is in the standalone folder)
or start it like 'ensure fh_towing' if the script is by it's own in the resources folder.



If you find any aspects unclear during the installation process, don't hesitate to join our Discord server mentioned above or post your questions on the cfx forum for assistance.
